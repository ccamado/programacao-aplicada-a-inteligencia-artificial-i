🔗 Conectando o Python à planilha real da Central de Avisos

📅 09/06/2026 ⏱ 50 min 🧪 Laboratório 🎯 Central de Avisos

🎯 O que você vai fazer nesta prática?

Nesta prática, você vai usar a planilha real da Central de Avisos para aprender a conectar o Python ao Google Sheets, ler a aba correta, identificar colunas, contar os tipos de aviso e criar um gráfico simples com os dados da turma.

Hoje você não vai inventar uma base. Você vai trabalhar com a base verdadeira que a turma já utiliza no projeto.

🧰 Bibliotecas da prática

  • Pandas para ler a planilha e trabalhar com tabela.
  • Numpy para organizar números em array.
  • Matplotlib para gerar um gráfico.

✅ Produto final

  • Link correto da aba AVISOS montado por você.
  • Planilha lida no Python.
  • Contagem dos tipos de aviso.
  • Gráfico simples com os dados da Central.

📘 Antes de começar: entenda a ideia principal

A planilha da Central de Avisos pode ter mais de uma aba, como AVISOS e HORARIO_SEMANAL. O Python precisa saber exatamente qual aba você quer ler. Por isso, não basta copiar qualquer link da planilha. Você precisa montar o link certo da aba.

Pense assim: a planilha é como um caderno com várias divisórias. O Python precisa saber qual divisória você quer abrir.
Passo 1

Importe as bibliotecas

Copie e execute este código:

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

O que esse código faz?

  • pandas as pd importa a biblioteca de tabelas.
  • numpy as np importa a biblioteca de números.
  • matplotlib.pyplot as plt importa a biblioteca de gráficos.
Se nada aparecer na tela, está tudo certo. Isso significa que as bibliotecas foram carregadas.
Passo 2

Abra a planilha e clique na aba certa

Agora abra a planilha da Central de Avisos no Google Sheets.

  1. Olhe para as abas na parte de baixo da planilha.
  2. Clique na aba AVISOS.
  3. Somente depois de abrir essa aba, observe o link do navegador.
Hoje você vai trabalhar com a aba AVISOS.
Passo 3

Encontre o ID da planilha

Com a aba AVISOS aberta, olhe para o link do navegador. Ele será parecido com este:

https://docs.google.com/spreadsheets/d/1ABCxyz1234567890/edit#gid=0

O ID da planilha é a parte que fica entre /d/ e /edit.

1ABCxyz1234567890

Agora registre no seu caderno ou notebook:

Meu ID da planilha é: ______________________________________
Passo 4

Encontre o gid da aba

No final do link aparece algo como #gid=0. Esse número identifica a aba que está aberta.

#gid=0

Se você mudar de aba, esse número pode mudar. Por isso, confirme de novo que a aba AVISOS está aberta.

O gid da aba AVISOS é: ______________________________________
Passo 5

Monte o link que o Python consegue ler

O Google Sheets usa um link comum para você navegar, mas o Python precisa de um link em formato CSV.

Use este modelo:

https://docs.google.com/spreadsheets/d/ID_DA_PLANILHA/export?format=csv&gid=ID_DA_ABA

Se o seu ID for 1ABCxyz1234567890 e o gid for 0, o link fica assim:

https://docs.google.com/spreadsheets/d/1ABCxyz1234567890/export?format=csv&gid=0
Meu link final da aba AVISOS é:

______________________________________________________________
Passo 6

Coloque o link no código

Agora copie e execute, trocando pelo seu link real:

url = "COLE_AQUI_O_LINK_FINAL_DA_ABA_AVISOS"

Exemplo:

url = "https://docs.google.com/spreadsheets/d/1ABCxyz1234567890/export?format=csv&gid=0"

Esse código cria uma variável chamada url e guarda nela o endereço da planilha que o Python vai ler.

Passo 7

Leia a planilha com Pandas

Copie e execute:

df = pd.read_csv(url)
print(df)

Como esse código funciona?

  • pd.read_csv(url) lê os dados do link.
  • df é o nome que você está dando para a tabela.
  • print(df) mostra a tabela na tela.
Se a tabela aparecer, isso significa que você conseguiu conectar o Python à planilha real da Central.
Passo 8

Olhe as primeiras linhas e veja os nomes das colunas

Copie e execute:

df.head()
print(df.columns)

Agora observe:

  • quais colunas existem;
  • como o nome da coluna de tipo está escrito;
  • se o nome está como tipo, Tipo ou outro formato.
Você precisa usar no código o nome da coluna exatamente como ele aparece na planilha.
Passo 9

Conte quantos avisos existem de cada tipo

Se a sua coluna se chamar tipo, copie e execute:

contagem_tipos = df["tipo"].value_counts()
print(contagem_tipos)

Se o nome da coluna for diferente, troque pelo nome real. Exemplo:

contagem_tipos = df["Tipo"].value_counts()
print(contagem_tipos)

O que esse código faz?

  • df["tipo"] pega só a coluna do tipo de aviso.
  • .value_counts() conta quantas vezes cada valor aparece.
Passo 10

Transforme as contagens em array com Numpy

Copie e execute:

valores = np.array(contagem_tipos.values)
print(valores)

Aqui você está:

  • pegando só os números da contagem;
  • transformando esses números em um array do Numpy.

Agora copie e execute também:

tipos = contagem_tipos.index
print(tipos)

Com isso, você terá:

  • tipos = nomes das categorias
  • valores = quantidades
Passo 11

Crie o gráfico

Copie e execute:

plt.figure(figsize=(8,5))
plt.bar(tipos, valores)
plt.title("Quantidade de avisos por tipo")
plt.xlabel("Tipo de aviso")
plt.ylabel("Quantidade")
plt.show()

O que cada linha faz?

  • plt.figure(figsize=(8,5)) cria a área do gráfico.
  • plt.bar(tipos, valores) cria as barras.
  • plt.title(...) define o título.
  • plt.xlabel(...) nomeia o eixo horizontal.
  • plt.ylabel(...) nomeia o eixo vertical.
  • plt.show() mostra o gráfico na tela.

  3. O gráfico ficou mais fácil de entender do que a tabela? Por quê?

📝 Resumo para o Caderno — Prática Guiada

  1. O Python pode ler uma planilha real do Google Sheets.
  2. Para isso, eu preciso escolher a aba correta.
  3. O ID da planilha identifica o arquivo inteiro.
  4. O gid identifica a aba que está aberta.
  5. O link normal precisa ser transformado em link CSV.
  6. O Pandas lê a planilha e transforma os dados em tabela.
  7. O value_counts conta quantas vezes cada tipo aparece.
  8. O Numpy organiza as quantidades em array.
  9. O Matplotlib transforma os números em gráfico.
  10. Com isso, eu consigo analisar a Central de Avisos com dados reais.

💻 Código completo da prática

import pandas as pd
import numpy as np
import matplotlib.pyplot as plt

url = "COLE_AQUI_O_LINK_FINAL_DA_ABA_AVISOS"

df = pd.read_csv(url)

print("Tabela completa:")
print(df)

print("\nPrimeiras linhas:")
print(df.head())

print("\nColunas da planilha:")
print(df.columns)

contagem_tipos = df["tipo"].value_counts()
print("\nContagem por tipo:")
print(contagem_tipos)

valores = np.array(contagem_tipos.values)
print("\nArray com as quantidades:")
print(valores)

tipos = contagem_tipos.index
print("\nCategorias:")
print(tipos)

plt.figure(figsize=(8,5))
plt.bar(tipos, valores)
plt.title("Quantidade de avisos por tipo")
plt.xlabel("Tipo de aviso")
plt.ylabel("Quantidade")
plt.show()
