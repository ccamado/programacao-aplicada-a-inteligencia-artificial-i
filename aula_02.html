🐼 Pandas inicial: abrir a base e fazer a primeira inspeção

📅 05/05/2026 ⏱ 50 min 🎯 Central de Avisos 📚 Programação Aplicada à Inteligência Artificial I

🎯 O que vamos aprender?

Nesta aula, vamos fazer o primeiro contato com o Pandas e entender como abrir uma tabela da Central de Avisos para observar linhas, colunas e valores. A proposta é enxergar a base como um conjunto organizado de registros.

💡 Aplicação no projeto: Usar o Pandas para visualizar as colunas da base e identificar observações iniciais sobre os dados da Central.

📘 Conteúdo principal

Conceito central: Primeira leitura técnica da base

Pandas é uma biblioteca do Python muito usada para manipular e analisar dados. Quando abrimos uma planilha ou tabela com Pandas, ela costuma aparecer em formato de DataFrame, que é como uma tabela com linhas e colunas. No projeto Central de Avisos, esse passo é importante porque permite ver se os registros estão completos, se as colunas fazem sentido e se existe algum padrão visível logo no início. A primeira inspeção não é uma análise profunda; é uma leitura inicial para responder perguntas como: quantas colunas existem? Quais nomes aparecem? Há dados vazios? Os formatos parecem consistentes? Essa observação prepara a turma para as próximas etapas de limpeza e organização.

🧠 Exemplos práticos

No quadro ou no notebook, vale destacar:
• coluna = tipo de informação;
• linha = um registro de aviso;
• DataFrame = tabela organizada para leitura;
• inspeção inicial = olhar a base antes de tentar analisá-la.

🚀 Microproduto da aula

Microproduto: tabela-resumo com nomes das colunas e 3 observações reais sobre a base.
