🐼 Primeiro contato com Pandas e inspeção inicial da base

📅 05/05/2026 ⏱ 50 min 🎯 Central de Avisos 📚 Programação Aplicada à IA I

🎯 O que vamos aprender?

Agora que já sabemos quais são as fontes da Central de Avisos, vamos iniciar a leitura das tabelas em Pandas para entender colunas, tipos de informação e problemas visíveis na base.

💡 Aplicação no projeto: Abrir a base, reconhecer colunas e fazer um diagnóstico inicial da qualidade dos dados da Central.

📦 Microproduto da aula

Tabela-resumo com nome das colunas, tipo de informação e 3 observações sobre a base.

🧾 Evidência de aprendizagem

Notebook de inspeção inicial da base da Central de Avisos.

📘 Conteúdo principal

Conceito central: Pandas é a biblioteca usada para abrir, visualizar e inspecionar tabelas, permitindo observar linhas, colunas, campos vazios e formatos diferentes.

Pandas é uma biblioteca do Python muito usada para manipular tabelas, chamadas de DataFrames. Em vez de olhar os dados de forma solta, Pandas ajuda a visualizar as primeiras linhas, contar colunas, perceber campos vazios e começar a entender a estrutura da base. No projeto Central de Avisos, isso significa abrir os dados e responder perguntas iniciais como: quantas colunas existem? Quais nomes aparecem? Há datas faltando? Os tipos de aviso estão consistentes? Essa inspeção não resolve tudo, mas prepara a turma para a limpeza e para as análises futuras.

📝 Resumo para o Caderno — Aula 02

  1. Pandas é uma biblioteca de Python para abrir e analisar tabelas.
  2. Uma tabela em Pandas é chamada de DataFrame.
  3. Inspecionar a base é observar colunas, linhas e tipos de informação.
  4. As primeiras linhas mostram como os dados foram registrados.
  5. Campos vazios podem indicar problema de preenchimento.
  6. Formatos diferentes no mesmo campo atrapalham a análise posterior.
  7. A inspeção inicial serve para conhecer a estrutura da base.
  8. Na Central de Avisos, isso ajuda a identificar dados úteis e falhas.
  9. Antes de limpar, precisamos olhar a base com atenção.
  10. Pandas ajuda a transformar observação em análise organizada.
