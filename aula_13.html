🤖 Scikit-learn e classificação simples aplicada à Central

📅 16/06/2026 ⏱ 50 min 🎯 Central de Avisos 📚 Programação Aplicada à Inteligência Artificial I

🎯 O que vamos aprender?

Nesta aula, vamos conhecer o papel do Scikit-learn e entender, em linguagem acessível, como um algoritmo pode ajudar a classificar dados.

💡 Aplicação no projeto: Ligar a ideia de algoritmo a situações simples de classificação dos avisos da Central.

📘 Conteúdo principal

Conceito central: Primeiro contato com aprendizado de máquina

Scikit-learn é uma biblioteca do Python voltada para algoritmos de aprendizado de máquina. Neste bimestre, o foco não é aprofundar matemática nem treinar modelos complexos, mas entender a ideia geral: um algoritmo pode aprender padrões a partir dos dados e ajudar a classificar informações. No projeto Central de Avisos, essa noção pode aparecer na organização por tipo, prioridade ou outra categoria. O importante é que a turma perceba a diferença entre uma regra fixa e um sistema que aprende com exemplos. A aula funciona como uma porta de entrada para o pensamento de classificação, mostrando que a preparação da base feita até agora é justamente o que torna possível avançar para usos mais inteligentes.

🧠 Exemplos práticos

Falas simples para o quadro:
• algoritmo = conjunto de passos;
• classificação = separar registros em grupos;
• base preparada ajuda o algoritmo a encontrar padrões;
• Scikit-learn é uma biblioteca usada para esse tipo de tarefa.
